# Implementacija Faznog Vokodera s Robotizacijom Šaptanjem i Promjenom Tonaliteta

Ovaj projekt predstavlja završni rad koji uključuje implementaciju faznog vokodera s dodatnim značajkama kao što su robotizacija, šaptanje i promjena tonaliteta. 
Cilj projekta bio je razviti aplikaciju unutar okruženja Bela koja može manipulirati zvučnim signalima koristeći napredne tehnike obrade signala.

## Upute za korištenje
1. Preuzmite i raspakirajte `Phase vocoder.zip`.
2. Prenesite zip datoteku unutar Bela IDE te pokrenite program.

## Ključne značajke
- **Fazni vokoder**: Implementacija vokodera za manipulaciju fazom zvučnog signala.
- **Robotizacija**: Dodavanje efekta robotizacije glasa.
- **Šaptanje**: Implementacija šaptajućeg efekta.
- **Promjena tonaliteta**: Mogućnost promjene tonaliteta ulaznog signala.
- **GUI**: Jednostavno sučelje za upravljanje efektima

## Tehnologije
- **Bela**: Platforma za obradu zvučnih signala u stvarnom vremenu.
- **C++/JavaScript**: Programski jezici korišteni za implementaciju algoritama.
